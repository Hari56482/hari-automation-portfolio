# Python Automation Portfolio — Damodhara Hari

Practical automation tools using **Python + Pandas** for:
- Cleaning Excel/CSV
- Converting PDF/bank statements to Excel/CSV
- Merging multiple monthly/weekly reports

## Scripts
| Task | Script | How to run (example) |
|---|---|---|
| Clean messy CSV/Excel | `scripts/clean_data.py` | `python scripts/clean_data.py` (expects `input.csv` next to it) |
| PDF → Excel/CSV | `scripts/pdf_to_excel.py` | `python scripts/pdf_to_excel.py` (expects `input.pdf`) |
| Merge many reports | `scripts/merge_reports.py` | `python scripts/merge_reports.py` (put files in `reports/`) |
